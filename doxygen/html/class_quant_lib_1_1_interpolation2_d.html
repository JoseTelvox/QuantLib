Detailed Description

base class for 2-D interpolations.

Classes derived from this class will provide interpolated values from two sequences of length and , representing the discretized values of the and variables, and a matrix representing the tabulated function values.

Warning
See the Interpolation class for information about the required lifetime of the underlying data.
