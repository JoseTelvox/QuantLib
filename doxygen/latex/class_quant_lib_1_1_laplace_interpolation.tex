\doxysubsection{Detailed Description}
Reconstruction of missing values using Laplace interpolation. We support an arbitrary number of dimensions n \texorpdfstring{$>$}{>}= 1 and non-\/equidistant grids. For n = 1 the method is identical to linear interpolation with flat extrapolation. Reference\+: Numerical Recipes, 3rd edition, ch. 3.\+8.
