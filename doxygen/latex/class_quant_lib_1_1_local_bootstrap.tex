\subsubsection*{template$<$class Curve$>$\newline
class Quant\+Lib\+::\+Local\+Bootstrap$<$ Curve $>$}
Localised-\/term-\/structure bootstrapper for most curve types. 

This algorithm enables a localised fitting for non-\/local interpolation methods.

As in the similar class (\doxylink{class_quant_lib_1_1_iterative_bootstrap}{Iterative\+Bootstrap}) the input term structure is solved on a number of market instruments which are passed as a vector of handles to \doxylink{class_quant_lib_1_1_bootstrap_helper}{Bootstrap\+Helper} instances. Their maturities mark the boundaries of the interpolated segments.

Unlike the \doxylink{class_quant_lib_1_1_iterative_bootstrap}{Iterative\+Bootstrap} class, the solution for each interpolated segment is derived using a local approximation. This restricts the risk profile s.\+t. the risk is localised. Therefore, we obtain a local IR risk profile whilst using a smoother interpolation method. Particularly good for the convex-\/monotone spline method.
