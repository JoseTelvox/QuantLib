\subsubsection*{template$<$class Interpolator1D$>$\newline
class Quant\+Lib\+::\+KInterpolated\+Yo\+YOptionlet\+Volatility\+Surface$<$ Interpolator1\+D $>$}
K-\/interpolated YoY optionlet volatility. 

The stripper provides curves in the T direction along each K. We don\textquotesingle{}t know whether this is interpolating or fitting in the T direction. Our K direction interpolations are not model fitting.

An alternative design would be a Fitted\+Yo\+YOptionlet\+Volatility\+Surface taking a model, e.\+g. \doxylink{class_quant_lib_1_1_s_a_b_r}{SABR} in the interest rate world. This could use the same stripping in the T direction along each K.
