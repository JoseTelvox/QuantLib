\doxysubsection{Detailed Description}
Market-\/model evolution description. 

This class stores\+:
\begin{DoxyEnumerate}
\item evolution\+Times = the times at which the rates need to be known,
\item rate\+Times = the times defining the rates that are to be evolved,
\item relevance\+Rates = which rates need to be known at each time.
\end{DoxyEnumerate}

This class is really just a tuple of evolution and rate times;
\begin{DoxyItemize}
\item there will be n+1 rate times expressing payment and reset times of forward rates.
\item there will be any number of evolution times.
\item we also store which part of the rates are relevant for pricing via relevance rates. The important part for the i-\/th step will then range from relevance\+Rates\mbox{[}i\mbox{]}.first to relevance\+Rates\mbox{[}i\mbox{]}.second. \doxylink{struct_quant_lib_1_1_default}{Default} values for relevance rates will be 0 and n.
