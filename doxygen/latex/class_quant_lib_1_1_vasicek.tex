\doxysubsection{Detailed Description}
Vasicek model class 

This class implements the \doxylink{class_quant_lib_1_1_vasicek}{Vasicek} model defined by   \[    dr_t = a(b - r_t)dt + \sigma dW_t ,
\] where $ a $, $ b $ and $ \sigma $ are constants; a risk premium $ \lambda $ can also be specified.
