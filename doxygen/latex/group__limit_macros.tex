\doxysubsubsection{Detailed Description}
Some compilers do not give an implementation of {\ttfamily \texorpdfstring{$<$}{<}limits\texorpdfstring{$>$}{>}} yet. For the code to be portable these macros should be used instead of the corresponding method of {\ttfamily std\+::numeric\+\_\+limits} or the corresponding macro defined in {\ttfamily \texorpdfstring{$<$}{<}limits.\+h\texorpdfstring{$>$}{>}}.
