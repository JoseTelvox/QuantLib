This defines a piecewise constant function which is RCLL and takes the values Y\mbox{[}0\mbox{]}, Y\mbox{[}1\mbox{]}, ... Y\mbox{[}n\mbox{]} on the intervals (-\/\textbackslash{}infty, X\mbox{[}0\mbox{]}), \mbox{[} X\mbox{[}1\mbox{]}, X\mbox{[}2\mbox{]} ), ... , \mbox{[} X\mbox{[}n-\/1\mbox{]}, \textbackslash{}infty) Normally Y.\+size() should be X.\+size() + 1. If more values for Y are given, they are ignored. If less values are given the last given value is kept the same for the remaining intervals. If X.\+size() is 0 a constant function taking the value Y\mbox{[}0\mbox{]} is evaluated.

\begin{DoxyWarning}{Warning}
If Y.\+size() is 0, an invalid access occurs. This condition is not checked for performance reasons. 
\end{DoxyWarning}
