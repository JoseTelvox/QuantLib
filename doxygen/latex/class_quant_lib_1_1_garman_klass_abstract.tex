\doxysubsection{Detailed Description}
Garman-\/\+Klass volatility model. 

This class implements a concrete volatility model based on high low formulas using the method of Garman and Klass in their paper "{}\+On the Estimation of the Security Price from \+Historical Data"{} at \href{http://www.fea.com/resources/pdf/a_estimation_of_security_price.pdf}{\texttt{ http\+://www.\+fea.\+com/resources/pdf/a\+\_\+estimation\+\_\+of\+\_\+security\+\_\+price.\+pdf}}

Volatilities are assumed to be expressed on an annual basis.
