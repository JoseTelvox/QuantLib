\doxysubsection{Detailed Description}
\subsubsection*{template$<$class RNG = Pseudo\+Random, class S = Statistics$>$\newline
class Quant\+Lib\+::\+MCBarrier\+Engine$<$ RNG, S $>$}
Pricing engine for barrier options using Monte Carlo simulation. 

Uses the Brownian-\/bridge correction for the barrier found in {\itshape  Going to Extremes\+: Correcting Simulation Bias in Exotic \doxylink{class_quant_lib_1_1_option}{Option} Valuation -\/ D.\+R. Beaglehole, P.\+H. Dybvig and G. Zhou Financial Analysts Journal; Jan/\+Feb 1997; 53, 1. pg. 62-\/68 } and {\itshape  Simulating path-\/dependent options\+: A new approach -\/ M. El Babsiri and G. Noel Journal of Derivatives; Winter 1998; 6, 2; pg. 65-\/83 }

\begin{DoxyRefDesc}{Test}
\item[\mbox{\hyperlink{test__test000109}{Test}}]the correctness of the returned value is tested by reproducing results available in literature. \end{DoxyRefDesc}
