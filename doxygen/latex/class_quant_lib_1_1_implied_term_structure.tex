\doxysubsection{Detailed Description}
Implied term structure at a given date in the future. 

The given date will be the implied reference date.

\begin{DoxyNote}{Note}
This term structure will remain linked to the original structure, i.\+e., any changes in the latter will be reflected in this structure as well.
\end{DoxyNote}
\begin{DoxyRefDesc}{Test}
\item[\mbox{\hyperlink{test__test000165}{Test}}]
\begin{DoxyItemize}
\item the correctness of the returned values is tested by checking them against numerical calculations.
\item observability against changes in the underlying term structure is checked. 
\end{DoxyItemize}\end{DoxyRefDesc}
