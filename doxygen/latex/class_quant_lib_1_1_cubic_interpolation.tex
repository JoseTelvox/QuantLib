\doxysubsection{Detailed Description}
Cubic interpolation between discrete points. 

\doxylink{class_quant_lib_1_1_cubic}{Cubic} interpolation is fully defined when the \$\{f\+\_\+i\}\$ function values at points \$\{x\+\_\+i\}\$ are supplemented with \$\{f\texorpdfstring{$^\wedge$}{\string^}\textquotesingle{}\+\_\+i\}\$ function derivative values.

Different type of first derivative approximations are implemented, both local and non-\/local. Local schemes (Fourth-\/order, \doxylink{class_quant_lib_1_1_parabolic}{Parabolic}, Modified \doxylink{class_quant_lib_1_1_parabolic}{Parabolic}, Fritsch-\/\+Butland, Akima, Kruger) use only \$f\$ values near \$x\+\_\+i\$ to calculate each \$f\texorpdfstring{$^\wedge$}{\string^}\textquotesingle{}\+\_\+i\$. Non-\/local schemes (Spline with different boundary conditions) use all \$\{f\+\_\+i\}\$ values and obtain \$\{f\texorpdfstring{$^\wedge$}{\string^}\textquotesingle{}\+\_\+i\}\$ by solving a linear system of equations. Local schemes produce \$C\texorpdfstring{$^\wedge$}{\string^}1\$ interpolants, while the spline schemes generate \$C\texorpdfstring{$^\wedge$}{\string^}2\$ interpolants.

Hyman\textquotesingle{}s monotonicity constraint filter is also implemented\+: it can be applied to all schemes to ensure that in the regions of local monotoniticity of the input (three successive increasing or decreasing values) the interpolating cubic remains monotonic. If the interpolating cubic is already monotonic, the Hyman filter leaves it unchanged preserving all its original features.

In the case of \$C\texorpdfstring{$^\wedge$}{\string^}2\$ interpolants the Hyman filter ensures local monotonicity at the expense of the second derivative of the interpolant which will no longer be continuous in the points where the filter has been applied.

While some non-\/linear schemes (Modified \doxylink{class_quant_lib_1_1_parabolic}{Parabolic}, Fritsch-\/\+Butland, Kruger) are guaranteed to be locally monotonic in their original approximation, all other schemes must be filtered according to the Hyman criteria at the expense of their linearity.

See R. L. Dougherty, A. Edelman, and J. M. Hyman, "{}\+Nonnegativity-\/, Monotonicity-\/, or Convexity-\/\+Preserving Cubic\+Spline and \+Quintic Hermite Interpolation"{} Mathematics Of Computation, v. 52, n. 186, April 1989, pp. 471-\/494.
