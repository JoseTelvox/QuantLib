\doxysubsection{Detailed Description}
United States calendars. 

Public holidays (see \href{https://www.opm.gov/policy-data-oversight/pay-leave/federal-holidays}{\texttt{ https\+://www.\+opm.\+gov/policy-\/data-\/oversight/pay-\/leave/federal-\/holidays}})\+: 
\begin{DoxyItemize}
\item Saturdays 
\item Sundays 
\item New \doxylink{namespace_quant_lib_a5dcda5470aff130981ac30f25d4244e7}{Year}\textquotesingle{}s \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, January 1st (possibly moved to Monday if actually on Sunday, or to Friday if on Saturday) 
\item Martin Luther King\textquotesingle{}s birthday, third Monday in January (since 1983) 
\item Presidents\textquotesingle{} \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day} (a.\+k.\+a. Washington\textquotesingle{}s birthday), third Monday in February 
\item Memorial \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, last Monday in May 
\item Juneteenth, June 19th (moved to Monday if Sunday or Friday if Saturday) 
\item Independence \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, July 4th (moved to Monday if Sunday or Friday if Saturday) 
\item Labor \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, first Monday in September 
\item Columbus \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, second Monday in October 
\item Veterans\textquotesingle{} \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, November 11th (moved to Monday if Sunday or Friday if Saturday) 
\item Thanksgiving \doxylink{namespace_quant_lib_a7354a4ac9189352870cbb0cb998e0bd4}{Day}, fourth Thursday in November 
\item Christmas, December 25th (moved to Monday if Sunday or Friday if Saturday) 
\end{DoxyItemize}
