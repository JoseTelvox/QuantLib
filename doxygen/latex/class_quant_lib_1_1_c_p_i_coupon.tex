\doxysubsection{Detailed Description}
Coupon paying the performance of a \doxylink{struct_quant_lib_1_1_c_p_i}{CPI} (zero inflation) index 

The performance is relative to the index value on the base date.

The other inflation value is taken from the ref\+Period\+End date with observation lag, so any roll/calendar etc. will be built in by the caller. By default this is done in the \doxylink{class_quant_lib_1_1_inflation_coupon}{Inflation\+Coupon} which uses Modified\+Preceding with fixing days assumed positive meaning earlier, i.\+e. always stay in same month (relative to reference\+Period\+End).

This is more sophisticated than an Indexed\+Cash\+Flow because it does date calculations itself.
