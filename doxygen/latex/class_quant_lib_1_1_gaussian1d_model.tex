\doxysubsection{Detailed Description}
One factor interest rate model interface class The only methods that must be implemented by subclasses are the numeraire and zerobond methods for an input array of state variable values. The variable \$y\$ is understood to be the standardized (zero mean, unit variance) version of the model\textquotesingle{}s original state variable \$x\$.

NTL support may be enabled by defining GAUSS1\+D\+\_\+\+ENABLE\+\_\+\+NTL in this file. For details on NTL see \href{http://www.shoup.net/ntl/}{\texttt{ http\+://www.\+shoup.\+net/ntl/}}

\begin{DoxyWarning}{Warning}
the variance of the state process conditional on \$x(t)=x\$ must be independent of the value of \$x\$ 
\end{DoxyWarning}
