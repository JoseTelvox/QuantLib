\doxysubsection{Detailed Description}
Derman-\/\+Kani-\/\+Ergener-\/\+Bardhan discretized option helper class. 

This class is used with the \doxylink{class_quant_lib_1_1_binomial_double_barrier_engine}{Binomial\+Double\+Barrier\+Engine} to implement the enhanced binomial algorithm of E.\+Derman, I.\+Kani, D.\+Ergener, I.\+Bardhan ("{}\+Enhanced Numerical Methods for Options with \+Barriers"{}, 1995)

\begin{DoxyNote}{Note}
This algorithm is only suitable if the payoff can be approximated linearly, e.\+g. is not usable for cash-\/or-\/nothing payoffs. 
\end{DoxyNote}
