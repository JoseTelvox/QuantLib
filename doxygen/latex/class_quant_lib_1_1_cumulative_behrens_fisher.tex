\doxysubsection{Detailed Description}
Cumulative (generalized) Behrens\+Fisher distribution. 

Exact analitical computation of the cumulative probability distribution of the linear combination of an arbitrary number (not just two) of T random variables of odd integer order. Adapted from the algorithm in\+:\begin{DoxyParagraph}{}
V. Witkovsky, Journal of Statistical Planning and Inference 94 (2001) 1-\/13
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
see also\+:
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
On the distribution of a linear combination of t-\/distributed variables; Glenn Alan Walker, Ph.\+D.\+thesis University of Florida 1977
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
\textquotesingle{}Convolutions of the T \doxylink{class_quant_lib_1_1_distribution}{Distribution}\textquotesingle{}; S. Nadarajah, D. K. Dey in Computers and Mathematics with Applications 49 (2005) 715-\/721
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
The last reference provides direct expressions for some of the densities when the linear combination of only two Ts is just an addition. It can be used for testing the results here.
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
Another available test on this algorithm stems from the realization that a linear convex ( $ \sum a_i=1$) combination of Ts of order one is stable in the distribution sense (but this result is often of no practical use because of its non-\/finite variance).
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
This implementation is for two or more T variables in the linear combination albeit these must be of odd order. The case of exactly two T of odd order is known to be a finite mixture of Ts but that result is not used here. Returns the cumulative probability of the resulting distribution. 

\begin{DoxyParagraph}{}
To obtain the cumulative probability the Gil-\/\+Pelaez theorem is applied\+:
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
First compute the characteristic function of the linear combination variable by multiplying the individual characteristic functions. Then transform back integrating the characteristic function according to the GP theorem; this is done here analytically feeding in the expression of the total characteristic function this\+:    \[ \int_0^{\infty}x^n e^{-ax}sin(bx)dx =
        (-1)^n \Gamma(n+1) \frac{sin((n+1)arctg2(-b/a))}
            {(\sqrt{a^2+b^2})^{n+1}}; for\,a>0,\,b>0
\] and for the first term I use\+:   \[    \int_0^{\infty} \frac{e^{-ax}sin(bx)}{x} dx = arctg2(b/a)
\] The GP complex integration is simplified thanks to the symetry of the distribution. 
\end{DoxyParagraph}
