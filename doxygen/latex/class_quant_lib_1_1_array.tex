\begin{Indent}\textbf{ Vector algebra}\par
{\em {\ttfamily v += x} and similar operation involving a scalar value are shortcuts for $ \forall i : v_i = v_i + x $

{\ttfamily v \texorpdfstring{$\ast$}{*}= w} and similar operation involving two vectors are shortcuts for $ \forall i : v_i = v_i \times w_i $

\begin{DoxyPrecond}{Precondition}
all arrays involved in an algebraic expression must have the same size.

1-\/D array used in linear algebra. 

This class implements the concept of vector as used in linear algebra. As such, it is {\bfseries{not}} meant to be used as a container -\/ {\ttfamily std\+::vector} should be used instead.
