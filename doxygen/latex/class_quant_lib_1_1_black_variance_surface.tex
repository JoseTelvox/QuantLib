\doxysubsection{Detailed Description}
Black volatility surface modelled as variance surface. 

This class calculates time/strike dependent Black volatilities using as input a matrix of Black volatilities observed in the market.

The calculation is performed interpolating on the variance surface. \doxylink{class_quant_lib_1_1_bilinear}{Bilinear} interpolation is used as default; this can be changed by the set\+Interpolation() method.
