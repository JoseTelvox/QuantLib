\doxysubsection{Detailed Description}
Credit event on a bond of a certain seniority(ies)/currency. 

Represents a credit event affecting all bonds with a given \textbackslash{} seniority and currency. It assumes that all such bonds suffer \textbackslash{} the event simultaneously. Some events affect all seniorities and this has to be encoded through a different set of events of the same event type. The event is an actual realization, not a contractual reference, as such it contains only an atomic type.
