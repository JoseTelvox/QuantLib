\doxysubsection{Detailed Description}
Zero-\/yield term structure. 

This abstract class acts as an adapter to \doxylink{class_quant_lib_1_1_yield_term_structure}{Yield\+Term\+Structure} allowing the programmer to implement only the {\ttfamily zero\+Yield\+Impl(\+Time)} method in derived classes.

\doxylink{struct_quant_lib_1_1_discount}{Discount} and forward are calculated from zero yields.

Zero rates are assumed to be annual continuous compounding.
