\doxysubsection{Detailed Description}
Normal distribution function. 

Given x, it returns its probability in a Gaussian normal distribution. It provides the first derivative too.

\begin{DoxyRefDesc}{Test}
\item[\mbox{\hyperlink{test__test000035}{Test}}]the correctness of the returned value is tested by checking it against numerical calculations.
