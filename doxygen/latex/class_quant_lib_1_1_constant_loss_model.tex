class Quant\+Lib\+::\+Constant\+Loss\+Model$<$ copula\+Policy $>$}
Constant\+Loss\+Latent\+Model interface for loss models. While it does not provide distribution type losses (e.\+g. expected tranche losses) because it lacks an integration algorithm it serves to allow pricing of digital type products like NTDs.
