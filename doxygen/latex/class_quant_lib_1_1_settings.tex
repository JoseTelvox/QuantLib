void Quant\+Lib\+::\+Settings\+::reset\+Evaluation\+Date (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Call this to reset the evaluation date to \doxylink{class_quant_lib_1_1_date_acdc9ad017a50abc0f14b8c4c77b3abe6}{Date\+::todays\+Date()} and allow it to change at midnight. It is equivalent to setting the evaluation date to Date(). This comes at the price of losing some performance, since the evaluation date is re-\/evaluated each time it is read.
