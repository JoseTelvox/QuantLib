This method returns the zero of the function $ f $, determined with the given accuracy $ \epsilon $; depending on the particular solver, this might mean that the returned $ x $ is such that  $ |f(x)| < \epsilon
$, or that $ |x-\xi| < \epsilon $ where $ \xi $ is the real zero.

An initial guess must be supplied, as well as two values $ x_\mathrm{min} $ and $ x_\mathrm{max} $ which must bracket the zero (i.\+e., either  $ f(x_\mathrm{min})
\leq 0 \leq f(x_\mathrm{max}) $, or  $f(x_\mathrm{max}) \leq 0 \leq f(x_\mathrm{min}) $ must be true).
