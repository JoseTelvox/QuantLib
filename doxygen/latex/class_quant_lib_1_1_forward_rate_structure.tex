\doxysubsection{Detailed Description}
Forward-\/rate term structure 

This abstract class acts as an adapter to \doxylink{class_quant_lib_1_1_yield_term_structure}{Yield\+Term\+Structure} allowing the programmer to implement only the {\ttfamily forward\+Impl(\+Time)} method in derived classes.

Zero yields and discounts are calculated from forwards.

\doxylink{class_quant_lib_1_1_forward}{Forward} rates are assumed to be annual continuous compounding.
