\doxysubsection{Detailed Description}
\subsubsection*{template$<$template$<$ class, class $>$ class derived\+Random\+LM, class copula\+Policy, class USNG = Sobol\+Rsg$>$\newline
class Quant\+Lib\+::\+Random\+LM$<$ derived\+Random\+LM, copula\+Policy, USNG $>$}
Base class for latent model monte carlo simulation. Independent of the copula type and the generator. Generates the factors and variable samples and determines event threshold but it is not responsible for actual event specification; thats the derived classes responsibility according to what they model. Derived classes need mainly to implement next\+Sample (Worker\+::next\+Sample in the multithreaded version) to compute the simulation event generated, if any, from the latent variables sample. They also have the accompanying event trait to specify.
