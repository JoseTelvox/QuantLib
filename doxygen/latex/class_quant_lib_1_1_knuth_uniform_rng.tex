\doxysubsection{Detailed Description}
Uniform random number generator. 

Random number generator by Knuth. For more details see Knuth, Seminumerical Algorithms, 3rd edition, Section 3.\+6. \begin{DoxyNote}{Note}
This is {\bfseries{not}} Knuth\textquotesingle{}s original implementation which is available at \href{http://www-cs-faculty.stanford.edu/~knuth/programs.html}{\texttt{ http\+://www-\/cs-\/faculty.\+stanford.\+edu/\texorpdfstring{$\sim$}{\string~}knuth/programs.\+html}}, but rather a slightly modified version wrapped in a C++ class. Such modifications did not affect the code but only the data structures used, which were converted to their standard C++ equivalents. 
\end{DoxyNote}
