\doxysubsection{Detailed Description}
Given a collection of vectors, w\+\_\+i, find a collection of vectors x\+\_\+i such that x\+\_\+i is orthogonal to w\+\_\+j for i != j, and \texorpdfstring{$<$}{<}x\+\_\+i, w\+\_\+i\texorpdfstring{$>$}{>} = \texorpdfstring{$<$}{<}w\+\_\+i, w\+\_\+i\texorpdfstring{$>$}{>}
