This constructor takes past fixings as a vector, defaulting to an empty vector representing an unseasoned option. This constructor expects {\itshape all} fixing dates to be provided, including those in the past, and to be already sorted. During the calculations, the option will compare them to the evaluation date to determine which are historic; it will then take as many values from all\+Past\+Fixings as needed and ignore the others. If not enough fixings are provided, it will raise an error.
