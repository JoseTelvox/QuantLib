\doxysubsection{Detailed Description}
\doxylink{namespace_quant_lib_ae2e788da35d2b866040a03b96227ee6e}{Statistics} tool. 

This class accumulates a set of data and returns their statistics (e.\+g\+: mean, variance, skewness, kurtosis, error estimation, percentile, etc.) based on the empirical distribution (no gaussian assumption)

It doesn\textquotesingle{}t suffer the numerical instability problem of \doxylink{class_quant_lib_1_1_incremental_statistics}{Incremental\+Statistics}. The downside is that it stores all samples, thus increasing the memory requirements.
