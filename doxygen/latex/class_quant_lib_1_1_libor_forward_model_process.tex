\doxysubsection{Detailed Description}
libor-\/forward-\/model process 

stochastic process of a libor forward model using the rolling forward measure incl. predictor-\/corrector step

References\+:

Glasserman, Paul, 2004, Monte Carlo Methods in Financial Engineering, Springer, Section 3.\+7

Antoon Pelsser, 2000, Efficient Methods for Valuing Interest \doxylink{namespace_quant_lib_a919ba3567cc89fca373f8d6b8e80126b}{Rate} Derivatives, Springer, 8

Hull, John, White, Alan, 1999, \doxylink{class_quant_lib_1_1_forward}{Forward} \doxylink{namespace_quant_lib_a919ba3567cc89fca373f8d6b8e80126b}{Rate} Volatilities, \doxylink{class_quant_lib_1_1_swap}{Swap} \doxylink{namespace_quant_lib_a919ba3567cc89fca373f8d6b8e80126b}{Rate} Volatilities and the Implementation of the \doxylink{class_quant_lib_1_1_libor}{Libor} Market Model (\href{http://www.rotman.utoronto.ca/~amackay/fin/libormktmodel2.pdf}{\texttt{ http\+://www.\+rotman.\+utoronto.\+ca/\texorpdfstring{$\sim$}{\string~}amackay/fin/libormktmodel2.\+pdf}})
