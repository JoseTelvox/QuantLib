\doxysubsection{Detailed Description}
Cox-\/\+Ingersoll-\/\+Ross model class. 

This class implements the Cox-\/\+Ingersoll-\/\+Ross model defined by   \[    dr(t) = k(\theta - r(t))dt + \sigma \sqrt{r(t)} dW(t)
\]
