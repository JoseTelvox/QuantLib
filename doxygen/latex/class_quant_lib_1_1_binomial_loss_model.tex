\doxysubsection{Detailed Description}
\subsubsection*{template$<$class LLM$>$\newline
class Quant\+Lib\+::\+Binomial\+Loss\+Model$<$ LLM $>$}
Binomial Defaultable \doxylink{class_quant_lib_1_1_basket}{Basket} \doxylink{class_quant_lib_1_1_loss}{Loss} Model\begin{DoxyParagraph}{}
Models the portfolio loss distribution by approximatting it to an adjusted binomial. Fits the two moments of the loss distribution through an adapted binomial approximation. This simple model allows for portfolio inhomogeneity with no excesive cost over the LHP.
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
See\+:
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
{\bfseries{Approximating Independent \doxylink{class_quant_lib_1_1_loss}{Loss} Distributions with an Adjusted Binomial \doxylink{class_quant_lib_1_1_distribution}{Distribution}}} , Dominic O\textquotesingle{}Kane, 2007 EDHEC RISK AND ASSET MANAGEMENT RESEARCH CENTRE 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
{\bfseries{Modelling single name and multi-\/name credit derivatives}} Chapter 18.\+5.\+2, Dominic O\textquotesingle{}Kane, Wiley Finance, 2008 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
The version presented here is adaptated to the multifactorial case by computing a conditional binomial approximation; notice that the Binomial is stable. This way the model can be used also in risk management models rather than only in pricing. The copula is also left undefined/arbitrary. 
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
LLM\+: \doxylink{class_quant_lib_1_1_loss}{Loss} Latent Model template parameter able to model default and loss.
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
The model is allowed and arbitrary copula, although initially designed for a Gaussian setup. If these exotic versions were not allowed the template parameter can then be dropped but the use of random recoveries should be added in some other way.
\end{DoxyParagraph}
