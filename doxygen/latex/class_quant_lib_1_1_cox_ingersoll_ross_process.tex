\doxysubsection{Detailed Description}
\doxylink{class_quant_lib_1_1_cox_ingersoll_ross}{Cox\+Ingersoll\+Ross} process class. 

This class describes the \doxylink{class_quant_lib_1_1_cox_ingersoll_ross}{Cox\+Ingersoll\+Ross} process governed by   \[    dx(t) = k (\theta - x(t)) dt + \sigma \sqrt{x(t)} dW(t).
\]

The process is discretized using the Quadratic Exponential scheme. For details see Leif Andersen, Efficient Simulation of the Heston Stochastic \doxylink{namespace_quant_lib_ae3abfa256de5aa2b506ab6bda014e4dc}{Volatility} Model.
