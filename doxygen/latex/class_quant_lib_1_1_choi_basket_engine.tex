\doxysubsection{Detailed Description}
Pricing engine for basket option on multiple underlyings. 

This class implements the pricing formula from "{}\+Sum of all Black-\/\+Scholes-\/\+Merton Models\+: An efficient Pricing Method for \+Spread, Basket and Asian Options"{}, Jaehyuk Choi, 2018 \href{https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2913048}{\texttt{ https\+://papers.\+ssrn.\+com/sol3/papers.\+cfm?abstract\+\_\+id=2913048}}

A Python implementation from the author of the paper is also available \href{https://github.com/PyFE/PyFENG}{\texttt{ https\+://github.\+com/\+Py\+FE/\+Py\+FENG}}
