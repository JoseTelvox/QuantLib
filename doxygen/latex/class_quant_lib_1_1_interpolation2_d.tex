\doxysubsection{Detailed Description}
base class for 2-\/D interpolations. 

Classes derived from this class will provide interpolated values from two sequences of length $ N $ and $ M $, representing the discretized values of the $ x $ and  $ y
$ variables, and a $ N \times M $ matrix representing the tabulated function values.

\begin{DoxyWarning}{Warning}
See the \doxylink{class_quant_lib_1_1_interpolation}{Interpolation} class for information about the required lifetime of the underlying data. 
\end{DoxyWarning}
