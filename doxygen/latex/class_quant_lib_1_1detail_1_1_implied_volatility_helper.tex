\doxysubsection{Detailed Description}
helper class for one-\/asset implied-\/volatility calculation 

The passed engine must be linked to the passed quote (see, e.\+g., \doxylink{class_quant_lib_1_1_vanilla_option}{Vanilla\+Option} to see how this can be achieved.)

\begin{DoxyNote}{Note}
this function is meant for developers of option classes so that they can implement an implied\+Volatility() method. 
\end{DoxyNote}
