\subsubsection*{template$<$class T$>$\newline
class Quant\+Lib\+::\+Clone$<$ T $>$}
cloning proxy to an underlying object 

When copied, this class will make a clone of its underlying object, which must provide a {\ttfamily clone()} method returning a std\+::auto\+\_\+ptr (or a std\+::unique\+\_\+ptr, depending on your configuration) to a newly-\/allocated instance.
