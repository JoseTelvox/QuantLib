\doxysubsection{Detailed Description}
Black variance term structure. 

This abstract class acts as an adapter to Vol\+Term\+Structure allowing the programmer to implement only the {\ttfamily black\+Variance\+Impl(\+Time, Real, bool)} method in derived classes.

\doxylink{namespace_quant_lib_ae3abfa256de5aa2b506ab6bda014e4dc}{Volatility} are assumed to be expressed on an annual basis.
