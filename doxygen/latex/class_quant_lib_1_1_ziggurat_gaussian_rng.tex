class Quant\+Lib\+::\+Ziggurat\+Gaussian\+Rng$<$ RNG $>$}
Gaussian random number generator. 

It uses the \doxylink{struct_quant_lib_1_1_ziggurat}{Ziggurat} transformation to return a normal distributed Gaussian deviate with average 0.\+0 and standard deviation of 1.\+0, from a random integer in the \mbox{[}0,0xffffffffffffffff\+ULL\mbox{]}-\/interval like.

For a more detailed description see the article "{}\+An Improved Ziggurat Method to Generate Normal Random Samples"{} by Jurgen A. Doornik (\href{https://www.doornik.com/research/ziggurat.pdf}{\texttt{ https\+://www.\+doornik.\+com/research/ziggurat.\+pdf}}).

The code here is inspired by the rust implementation in \href{https://github.com/rust-random/rand/blob/d42daabf65a3ceaf58c2eefc7eb477c4d5a9b4ba/rand_distr/src/normal.rs}{\texttt{ https\+://github.\+com/rust-\/random/rand/blob/d42daabf65a3ceaf58c2eefc7eb477c4d5a9b4ba/rand\+\_\+distr/src/normal.\+rs}} and \href{https://github.com/rust-random/rand/blob/d42daabf65a3ceaf58c2eefc7eb477c4d5a9b4ba/rand_distr/src/utils.rs}{\texttt{ https\+://github.\+com/rust-\/random/rand/blob/d42daabf65a3ceaf58c2eefc7eb477c4d5a9b4ba/rand\+\_\+distr/src/utils.\+rs}}.
