\doxysubsection{Detailed Description}
At-\/the-\/money swaption-\/volatility matrix. 

This class provides the at-\/the-\/money volatility for a given swaption by interpolating a volatility matrix whose elements are the market volatilities of a set of swaption with given option date and swap\+Length.

The volatility matrix {\ttfamily M} must be defined so that\+:
\begin{DoxyItemize}
\item the number of rows equals the number of option dates;
\item the number of columns equals the number of swap tenors;
\item {\ttfamily M\mbox{[}i\mbox{]}\mbox{[}j\mbox{]}} contains the volatility corresponding to the {\ttfamily i}-\/th option and {\ttfamily j}-\/th tenor. 
\end{DoxyItemize}
