class Quant\+Lib\+::\+Polar\+Student\+TRng$<$ URNG $>$}
Student t random number generator. 

Polar transformation based Student T random number generator. See "{}\+Polar Generation of Random Variates With the t-\/\+Distribution"{}, Ralph W. Bailey, April 1994, in Mathematics of Computation, Vol 62-\/206 page 779. The one implemented here is a variant of this algorithm from "{}\+Random  \+Number Generation and Monte Carlo Methods"{}, Springer, 2003, page 185. Using a uniform RNG on a \mbox{[}-\/1,1\mbox{]} support, the extra call to the uniform generator (used for the sign of the sample) is avoided.
