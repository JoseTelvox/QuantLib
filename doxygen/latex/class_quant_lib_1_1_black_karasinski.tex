\doxysubsection{Detailed Description}
Standard Black-\/\+Karasinski model class. 

This class implements the standard Black-\/\+Karasinski model defined by   \[    d\ln r_t = (\theta(t) - \alpha \ln r_t)dt + \sigma dW_t,
\] where $ alpha $ and $ sigma $ are constants.
