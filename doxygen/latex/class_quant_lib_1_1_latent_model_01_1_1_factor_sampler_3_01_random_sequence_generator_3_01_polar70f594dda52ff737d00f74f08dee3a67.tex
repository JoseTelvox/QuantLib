Specialization for direct T samples generation. 

\begin{DoxyParagraph}{}
The PolarT is a rejection algorithm so do not use it within a multithreaded simulation. The \doxylink{class_quant_lib_1_1_random_sequence_generator}{Random\+Sequence\+Generator} class does not admit heterogeneous distribution samples so theres a trick here since the template parameter is not what it is used internally. 
\end{DoxyParagraph}
