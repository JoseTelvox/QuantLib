\subsubsection*{template$<$class T, class Container = std\+::map$<$\+Date, T$>$$>$\newline
class Quant\+Lib\+::\+Time\+Series$<$ T, Container $>$}
Container for historical data. 

This class acts as a generic repository for a set of historical data. Any single datum can be accessed through its date, while sets of consecutive data can be accessed through iterators.

\begin{DoxyPrecond}{Precondition}
The {\ttfamily Container} type must satisfy the requirements set by the C++ standard for associative containers. 
\end{DoxyPrecond}
