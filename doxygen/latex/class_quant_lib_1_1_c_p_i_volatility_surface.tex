zero inflation (i.\+e. CPI/\+RPI/\+HICP/etc.) volatility structures 

Abstract interface. \doxylink{struct_quant_lib_1_1_c_p_i}{CPI} volatility is always with respect to some base date. Also deal with lagged observations of an index with a (usually different) availability lag.
