\doxysubsection{Detailed Description}
\doxylink{namespace_quant_lib_a919ba3567cc89fca373f8d6b8e80126b}{Rate} helper for bootstrapping over overnight-\/ibor basis swaps. 

The swap is assumed to pay base\+Index + basis and receive other\+Index. This helper can be used to bootstrap the forecast curve for other\+Index; base\+Index will need an existing forecast curve. An exogenous discount curve can be passed; if not, the overnight-\/index curve will be used.
