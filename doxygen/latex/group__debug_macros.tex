\doxysubsubsection{Detailed Description}
For debugging purposes, macros can be used to output information about the code being executed. Instrumenting code as in\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{namespace\ }Foo\ \{}
\DoxyCodeLine{}
\DoxyCodeLine{\ \ \ \ \textcolor{keywordtype}{int}\ bar(\textcolor{keywordtype}{int}\ i)\ \{}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga903ea9f035842ee3560fd787c84887d7}{QL\_TRACE\_ENTER\_FUNCTION}};}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga7d82587448e3f5531f79cfb215b905af}{QL\_TRACE\_VARIABLE}}(i);}
\DoxyCodeLine{}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \textcolor{keywordflow}{if}\ (i\ ==\ 42)\ \{}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga4b80d6b6b9d0d17493ac81428934f1c0}{QL\_TRACE\_LOCATION}};}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga456b5371fdc64479d393a04b505df5d0}{QL\_TRACE}}(\textcolor{stringliteral}{"{}Right\ answer,\ but\ no\ question"{}});}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \}\ \textcolor{keywordflow}{else}\ \{}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga4b80d6b6b9d0d17493ac81428934f1c0}{QL\_TRACE\_LOCATION}};}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga456b5371fdc64479d393a04b505df5d0}{QL\_TRACE}}(\textcolor{stringliteral}{"{}Wrong\ answer"{}});}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ i\ *=\ 2;}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \}}
\DoxyCodeLine{}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga7d82587448e3f5531f79cfb215b905af}{QL\_TRACE\_VARIABLE}}(i);}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \mbox{\hyperlink{group__debug_macros_ga33c013fea8c974f6781441deaa3ea53b}{QL\_TRACE\_EXIT\_FUNCTION}};}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \textcolor{keywordflow}{return}\ i;}
\DoxyCodeLine{\ \ \ \ \}}

\end{DoxyCode}
 will output a trace like the following when the code is run\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{trace[3]:\ Entering\ \textcolor{keywordtype}{int}\ Foo::bar(\textcolor{keywordtype}{int})}
\DoxyCodeLine{trace[3]:\ i\ =\ 21}
\DoxyCodeLine{trace[3]:\ At\ line\ 16\ in\ tracing\_example.cpp}
\DoxyCodeLine{trace[3]:\ Wrong\ answer}
\DoxyCodeLine{trace[3]:\ i\ =\ 42}
\DoxyCodeLine{trace[3]:\ Exiting\ \textcolor{keywordtype}{int}\ Foo::bar(\textcolor{keywordtype}{int})}

\end{DoxyCode}
 (the actual output will depend on the compiler and the file names). A word of warning must be added\+: adding so much tracing to your code might degrade its readability.
