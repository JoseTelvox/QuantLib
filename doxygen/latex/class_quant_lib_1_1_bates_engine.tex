\doxysubsection{Detailed Description}
Bates model engines based on Fourier transform. 

this classes price european options under the following processes


\begin{DoxyEnumerate}
\item Jump-\/\+Diffusion with Stochastic \doxylink{namespace_quant_lib_ae3abfa256de5aa2b506ab6bda014e4dc}{Volatility}

\[\begin{array}{rcl}
dS(t, S)  &=& (r-d-\lambda m) S dt +\sqrt{v} S dW_1 + (e^J - 1) S dN \\
dv(t, S)  &=& \kappa (\theta - v) dt + \sigma \sqrt{v} dW_2 \\
dW_1 dW_2 &=& \rho dt
\end{array}
\]
\end{DoxyEnumerate}

N is a Poisson process with the intensity  $ \lambda
$. When a jump occurs the magnitude J has the probability density function $ \omega(J) $.

1.\+1 Log-\/\+Normal Jump Diffusion\+: \doxylink{class_quant_lib_1_1_bates_engine}{Bates\+Engine}

Logarithm of the jump size J is normally distributed    \[\omega(J) = \frac{1}{\sqrt{2\pi \delta^2}}
            \exp\left[-\frac{(J-\nu)^2}{2\delta^2}\right]
\]

1.\+2 Double-\/\+Exponential Jump Diffusion\+: \doxylink{class_quant_lib_1_1_bates_double_exp_engine}{Bates\+Double\+Exp\+Engine}

The jump size has an asymmetric double exponential distribution       \[\begin{array}{rcl}
\omega(J)&=&  p\frac{1}{\eta_u}e^{-\frac{1}{\eta_u}J} 1_{J>0}
            + q\frac{1}{\eta_d}e^{\frac{1}{\eta_d}J} 1_{J<0} \\
p + q &=& 1
\end{array}
\]


\begin{DoxyEnumerate}
\item Stochastic \doxylink{namespace_quant_lib_ae3abfa256de5aa2b506ab6bda014e4dc}{Volatility} with Jump Diffusion and Deterministic Jump Intensity

\[\begin{array}{rcl}
dS(t, S)  &=& (r-d-\lambda m) S dt +\sqrt{v} S dW_1 + (e^J - 1) S dN \\
dv(t, S)  &=& \kappa (\theta - v) dt + \sigma \sqrt{v} dW_2 \\
d\lambda(t) &=& \kappa_\lambda(\theta_\lambda-\lambda) dt \\
dW_1 dW_2 &=& \rho dt
\end{array}
\]
\end{DoxyEnumerate}

2.\+1 Log-\/\+Normal Jump Diffusion with Deterministic Jump Intensity \doxylink{class_quant_lib_1_1_bates_det_jump_engine}{Bates\+Det\+Jump\+Engine}

2.\+2 Double-\/\+Exponential Jump Diffusion with Deterministic Jump Intensity \doxylink{class_quant_lib_1_1_bates_double_exp_det_jump_engine}{Bates\+Double\+Exp\+Det\+Jump\+Engine}
