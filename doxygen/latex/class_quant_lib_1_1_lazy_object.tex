This method performs all needed calculations by calling the {\itshape {\bfseries{perform\+Calculations}}} method.

\begin{DoxyWarning}{Warning}
Objects cache the results of the previous calculation. Such results will be returned upon later invocations of {\itshape {\bfseries{calculate}}}. When the results depend on arguments which could change between invocations, the lazy object must register itself as observer of such objects for the calculations to be performed again when they change.

Should this method be redefined in derived classes, \doxylink{class_quant_lib_1_1_lazy_object_a5244c7f83592375a0ea9048296435fd1}{Lazy\+Object\+::calculate()} should be called in the overriding method. 
\end{DoxyWarning}

This method causes the object to forward the first notification received, and discard the others until recalculated; the rationale is that observers were already notified, and don\textquotesingle{}t need further notifications until they recalculate, at which point this object would be recalculated too. After recalculation, this object would again forward the first notification received.

Although not always correct, this behavior is a lot faster and thus is the current default.
